Detailed Description

Stores an IPv4 address.

This class stores an IPv4 address and has various getters/setters for compatibility with various commonly used IP storage formats.

Internally this class stores the IP as an in_addr struct, thus if speed is a concern, the in_addr getter/setters should be used. The other getter/setters convert to/from in_addr.
